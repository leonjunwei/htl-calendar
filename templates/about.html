The Team


This app was made with love and care by Leon Lam (Olin College '19) and Emily Yeh (Olin College '19) for Olin's Spring 2017 Hacking the Library course. The calendar is still a work in progress, but we hope it will be improved and useful to Oliners in the future.

In terms of future work, it would be great if the following features were added to this app:
♦ Manually or automatically import/export events from mailing lists
♦ Search database for events by time ("after 6:00", "before 23:00", etc.)
♦ Search database for events by tags ("carpe", "dance", etc.)
